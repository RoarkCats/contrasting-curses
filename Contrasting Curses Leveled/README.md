# <img src="pack.png" alt="drawing" width="24"/> Contrasting Curses: Leveled

### Addon Note:
This pack is an **addon** for [Contrasting Curses](https://modrinth.com/datapack/contrasting-curses) that **enables leveled curses**!  
Using this pack in any form on it's own will result in *very broken* curses and functionality, both are required.

## Contrasting Curses

📕 Adds a collection of 20 unique curses spread across all types of equipment equally with interesting tradeoffs crafted in to encourage interaction with cursed gear. While all curses are generally negative, some may have useful niche applications or specific buffs when playing around them.

➕ Most curses stack and support multiple levels, allowing for all sorts of unhinged and cursed fun! Combine curses in an anvil to increase their level like any other enchant or find them naturally on gear! If you'd like to play *without* leveled curses, do *not* install this addon, instead just use the base [Contrasting Curses](https://modrinth.com/datapack/contrasting-curses) pack on its own.

## Questions

> **How do I install and use this datapack?**
> 
> Simply place the datapack zip file in the `datapacks` folder of your world alongside [Contrasting Curses](https://modrinth.com/datapack/contrasting-curses) and rejoin. You can also drag them into the datapacks screen during world creation, just make sure to enable them on the right side with Leveled on top. Alternatively, you can use any global datapack loader mod or the mod packaged version!

> **Help! My curses aren't leveled!**
> 
> Ensure that you have installed and running **both** the base [Contrasting Curses](https://modrinth.com/datapack/contrasting-curses) and the Leveled addon. If you're using the datapacks, make sure that Contrasting Curses: Leveled loads **after** the base pack, that means it should show up above base Contrasting Curses, with higher priority. If you're still having issues, please reach out on Discord or GitHub.

> **How can I know the max level of curses in-game?**
>
> Various mods allow you to see the max level of enchantments such as [Enchantment Insights](https://modrinth.com/project/xLSpHE43) or JEI / EMI with their various enchantment addons. These also usually allow you to view supported items and other incompatible enchants!

> **Can I add or remove this addon from my world?**
>
> If this addon is added later, existing level 1 enchants will turn into their leveled variants at level 1. If removed, over-leveled enchants will remain at their level so long as the base pack remains installed.
